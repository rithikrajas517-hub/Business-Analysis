# Business Sales Analysis Dashboard - Power BI

## Overview
This project presents an interactive Business Sales Analysis Dashboard developed using Power BI to analyze sales performance and generate actionable business insights. The dashboard helps visualize key business metrics and supports data-driven decision-making through interactive reports and visual storytelling.

## Dashboard Features
- Stock availability monitoring
- Quantity purchased analysis
- Median sales tracking
- Country-wise sales performance analysis
- Customer loyalty points analysis
- Product-wise purchase analysis
- Sales trend analysis over time
- Profit overview

## Key Insights
- Identified purchasing trends across multiple countries
- Analyzed customer loyalty and buying behavior
- Monitored sales fluctuations over time
- Evaluated product performance based on purchase quantity
- Tracked important KPIs for business performance

## Technologies Used
- Power BI
- DAX
- Microsoft Excel
- Data Cleaning
- Data Visualization
- Data Analysis

## Dashboard Components
- KPI Cards
- Bar Charts
- Pie Charts
- Trend Line Charts
- Interactive Filters

## Project Outcome
The dashboard enables quick understanding of business performance by transforming raw sales data into meaningful insights, helping businesses identify trends and make informed decisions.
